# 🏦 Bank Management System using Python

## 📌 Project Overview

The Bank Management System is a Python-based application developed to simulate core banking operations. The system enables users to securely create bank accounts, authenticate using login credentials, perform financial transactions, and maintain transaction records. It also includes OTP verification and account security features to ensure a safe banking experience.

---

## ✨ Key Features

- User Registration
- Secure User Login
- Email OTP Verification
- Deposit Money
- Withdraw Money
- Fund Transfer
- Balance Enquiry
- Transaction History
- PIN Reset
- Account Lock & Unlock
- Secure JSON Data Storage

## 🚀 How to Run

1. Clone the repository.
2. Open the project in Jupyter Notebook.
3. Install the required libraries.
4. Run all notebook cells.
5. Register a new account and start using the application.
